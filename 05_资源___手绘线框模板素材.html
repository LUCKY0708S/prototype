目前还是很多人喜欢线框原型草图，这仍然是一个进行WEB或者是APP原型设计的可靠的真实的方法。很多应用程序就开始于纸上的草图。于是我们希望能够打印一些移动设备模板设计并且到纸上进行手工绘图设计。

关于手绘线框原型草图

SneakPeekit

在线移动网页计打印工具是网页设计师，开发商或图形设计师的必备工具之一，设计师可以通过该平台提供的大量的模版来设计网页草图并且可以快速的打印出来。SneakPeekit支持网页设计、移动终端网页设计、iPad网页设计，提供多种格式的线框，包含1140px、960px、768PX、640px、480px、320px。

http://sneakpeekit.com

Sketchsheets

该网站的上所有手机模型都是png或者是pdf格式的，各位UI/UX设计师及产品经理可以则需下载想要的模板打印出来即可。

http://sketchsheets.com

Interface Sketch

UI/UX设计师及产品经理必备的APP手绘草图模板。Interface Sketch是一个免费提供给大家使用的设计草图模板，提供了PDF格式的设计模板。

http://www.interfacesketch.com

UI Stencils

UI Stencils工具分为两大类：UI元素模板，空白草稿板。它们既可以单独使用，又可以混搭在一起使用。UI元素模板分为 iPhone\iPad\Android\Windows Phone\Website 这五个品种。

http://www.uistencils.com
